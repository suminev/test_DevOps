# Тестовое задание на позицию DevOps

1. Создать скрипт /usr/local/bin/monitor_test.sh
2. Выдать права на исполнение
3. Создать файл сервиса /etc/systemd/system/monitor_test.service
4. Создать файл таймера /etc/systemd/system/monitor_test.timer
5. Запустить сервис monitor_test.timer и добавить его в автозагрузку

Примечание:
* Должны быть выданы права на чтение и запись в директорию /var/log/ или файл /var/log/monitoring.log
